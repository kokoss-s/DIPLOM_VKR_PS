Программная система для расчёта параметров развития пожара на производственном объекте в Тульской области

С помощью данного онлайн ресурса сотрудники главного управления МЧС России по Тульской области могут выполнять расчёты параметров развития пожара на производственном объекте

Перед началом выполнения расчёта параметров распространения пожара, необходимо подготовить данные о текущем пожаре, такие как: время с момента начала пожара и тип объекта, на котором начался пожар.

Для расчёта параметров перейдите на страницу расчётов, выберите формулу для расчёта параметров развития пожара и введите время с момента начала пожара, введите угол распространения пожара (только для угловой формулы) и выберите тип объекта, на котором начался пожар.

Результаты расчёта будут предоставлены после нажатия кнопки "Рассчитать" на отдельной странице, подготовленой для печати.
